# 去哪儿M站

## Project setup
```
npm i
```

### 启动开发环境
```
npm run serve
```

### 打包项目到/dist目录
```
npm run build
```


### webpack打包vue.config.js配置详情
See [Configuration Reference](https://cli.vuejs.org/config/).


## **重要说明,该项目没有线上接口,mock数据都放在/public/mock/文件夹下.所以打包后通过ajax获取的数据无法显示,在开发环境可以获取数据.如需打包后看见数据需要后端提供数据接口**
